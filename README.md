# 📊 Finance Analysis Dashboard | Power BI

An interactive Finance Analysis Dashboard developed in Microsoft Power BI to monitor financial performance, customer behavior, transaction trends, fees, and taxes. The dashboard provides real-time business insights through dynamic KPIs, interactive visualizations, and drill-down analysis to support data-driven decision-making.

---

## 📌 Project Overview

This project was developed to help a financial organization analyze its transaction data through an interactive Power BI dashboard. It enables stakeholders to monitor business performance, customer segments, transaction status, regional performance, and financial KPIs in a centralized reporting solution.

---

## 🎯 Business Requirements

The dashboard was designed to address the following business challenges:

- Monitor overall transaction growth and financial performance.
- Analyze monthly transaction trends.
- Compare successful, failed, and pending transactions.
- Identify high-performing customer segments.
- Evaluate state-wise financial performance.
- Analyze profitability by transaction type.
- Understand customer demographics (Gender).
- Track Year-over-Year (YoY) performance.
- Monitor operational fees and taxes.
- Enable dynamic filtering by Year, Occupation, Category, and Measure.

---

## 📊 Key Performance Indicators (KPIs)

- 💰 Total Amount
- 🔄 Total Transactions
- 📈 Average Transaction Value
- 💳 Total Fees
- 🧾 Total Tax
- 📊 Year-over-Year (YoY) Growth
